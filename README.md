# React Pokédex

A WEB2 Assignment 2 solution built with React, Vite, React Router and PokéAPI.

## Features

- Pokédex page with paginated Pokémon list
- Detail page for each Pokémon
- About page
- GitHub Pages deployment workflow
- Hash-based routing for GitHub Pages

## Run locally

```bash
npm install
npm run dev
```

## Build and preview

```bash
npm run build
npm run preview
```
